# 📋 Инструкция для Менеджера

## 🔐 Авторизация

### Первый вход
1. Найдите бота в Telegram по нику/ссылке
2. Нажмите кнопку "Start"
3. Введите пароль, выданный администратором
4. После успешной авторизации вы получите доступ к менеджерскому меню

### Повторный вход
- Бот запоминает вашу авторизацию
- При последующих запусках вы сразу попадете в меню

## 🎯 Основные функции Менеджера

### 📝 Создание заказа

#### Процесс создания
1. Нажмите "📝 Создать заказ"
2. Заполните информацию о клиенте:
   - **Имя клиента** - полное имя заказчика
   - **Телефон** - контактный номер для связи
   - **Адрес** - точный адрес уборки
3. Выберите город из списка
4. Укажите тип уборки:
   - 🧹 Регулярная уборка
   - 🧼 Глубокая уборка  
   - 🏗️ Послестроительная уборка
   - 🪟 Уборка окон
5. Укажите дату и время в формате: ДД.ММ.ГГГГ ЧЧ:ММ
6. Укажите продолжительность в часах (например: 2.5)
7. Укажите стоимость заказа в рублях
8. Ответьте на вопросы об оборудовании и средствах
9. Добавьте примечания (опционально)

#### После создания заказа
- Заказ автоматически появляется в теме города для клинеров
- Вы получаете подтверждение создания
- Клинеры могут видеть и брать заказ

### 📋 Управление заказами

#### Мои заказы
1. Нажмите "📋 Мои заказы"
2. Просмотрите список ваших заказов со статусами:
   - 🆕 Новый - еще не взят клинером
   - 👤 Назначен - клинер взят заказ
   - 🔄 В работе - клинер выполняет уборку
   - ✅ Выполнен - работа завершена
   - ❌ Отменен - заказ отменен

#### Взаимодействие с клинером
- Через бота вы получаете уведомления о статусах заказов
- При взятии заказа клинером вы получаете его контакты
- Можете связаться с клинером для уточнения деталей

### 💰 Финансовые операции

#### Выставление счета клиенту
1. Нажмите "💰 Выставить счет"
2. Выберите заказ из списка
3. Система автоматически создаст:
   - PDF счет с деталями заказа
   - QR-код для быстрой оплаты
4. Отправьте эти документы клиенту
5. Клиент может оплатить по QR-коду или реквизитам

#### Оплата услуг клинера
1. Нажмите "💳 Оплатить клинеру"
2. Выберите выполненный заказ
3. Система покажет:
   - Сумму к оплате клинеру
   - Реквизиты клинера
   - Детали заказа
4. Произведите оплату по указанным реквизитам
5. Подтвердите платеж в системе

### 📊 Статистика

#### Ваша личная статистика
- **Всего заказов** - общее количество созданных заказов
- **Выполнено** - успешно завершенные заказы
- **Процент выполнения** - эффективность работы
- **Общая выручка** - сумма всех выполненных заказов
- **Средний чек** - средняя стоимость заказа

#### Мониторинг эффективности
- Отслеживайте свои показатели
- Сравнивайте результаты по периодам
- Оптимизируйте процесс создания заказов

### 📄 Реквизиты компании

#### Просмотр реквизитов
1. Нажмите "📄 Реквизиты"
2. Вы увидите:
   - Название компании
   - ИНН
   - Банковские реквизиты
   - Контактная информация

#### Использование реквизитов
- Для выставления счетов клиентам
- Для юридического оформления
- Для предоставления клиентам

2. Изучите доступные команды
3. Свяжитесь с администратором при проблемах

### 🚨 Решение проблем

#### Клиент не оплатил
- Свяжитесь с клиентом
- Отправьте напоминание со счетом
- Предложите альтернативные способы оплаты

- Уточните причины задержки
- При необходимости переназначьте заказ

#### Технические проблемы
- Перезапустите бот
- Проверьте интернет соединение
- Свяжитесь с администратором

## 💡 Советы по эффективности

### 📋 Организация работы
1. **Создавайте заказы заранее** - это дает клинерам время на планирование
2. **Собирайте полную информацию** - точные данные снижают количество вопросов
3. **Используйте примечания** - фиксируйте особые требования клиентов
4. **Проверяйте контакты** - убедитесь что телефоны клиентов верны

### 💰 Финансовая оптимизация
1. **Правильно оценивайте стоимость** - учитывайте сложность и длительность
2. **Своевременно выставляйте счета** - ускоряет получение оплаты
3. **Контролируйте оплату клинеров** - поддерживайте хорошую репутацию

### 🤂 Коммуникация
1. **Будьте доступны** - отвечайте на сообщения быстро
