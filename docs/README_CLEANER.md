# 📋 Инструкция для Клинера

## 🔐 Начало работы

### Регистрация
1. Администратор добавит вас в систему
2. Вы получите уведомление от бота
3. Бот автоматически определит ваш город
4. После регистрации вы получите доступ к меню клинера

### Авторизация
- Бот автоматически узнает вас по Telegram ID
- Дополнительная авторизация не требуется
- Убедитесь что ваш профиль активен

## 🎯 Основные функции Клинера

### 📋 Поиск заказов

#### Доступные заказы
1. Нажмите "📋 Доступные заказы"
2. Вы увидите список новых заказов в вашем городе:
   - 🆕 Заказ #XXX
   - 👤 Клиент: Имя клиента
   - 📍 Адрес: Адрес уборки
   - 🧹 Тип: Тип уборки
   - 🕐 Время: Дата и время
   - ⏱ Длительность: X часов
   - 💰 Цена: X руб.

#### Фильтрация заказов
- Заказы показываются только для вашего города
- Сортировка по времени выполнения
- Обновление списка в реальном времени

#### Взятие заказа
1. Выберите интересующий заказ
2. Нажмите кнопку "✅ Взять заказ"
3. После подтверждения вы получите:
   - Полные детали заказа
   - Контактную информацию клиента
   - Контакт менеджера для связи

### 🏠 Мои заказы

#### Управление заказами
1. Нажмите "🏠 Мои заказы"
2. Вы увидите статус ваших заказов:
   - 👤 Назначен - заказ взят вами
   - 🔄 В работе - вы выполняете уборку
   - ✅ Выполнен - работа завершена
   - ❌ Отменен - заказ отменен

#### Детали заказа
- Полная информация о клиенте
- Адрес и контактные данные
- Требования к уборке
- Время и стоимость

### 📸 Загрузка фотографий

#### Процесс загрузки
1. Нажмите "📸 Загрузить фото"
2. Выберите заказ из списка активных
3. Система покажет статус фото:
   - ✅ Фото 'ДО' загружено
   - ❌ Фото 'ДО' не загружено
   - ✅ Фото 'ПОСЛЕ' загружено
   - ❌ Фото 'ПОСЛЕ' не загружено

#### Требования к фото
**Фото 'ДО':**
- Сделайте перед началом работ
- Покажите общее состояние помещения
- Зафиксируйте проблемные зоны

**Фото 'ПОСЛЕ':**
- Сделайте после завершения уборки
- Покажите результат работ
- Сравните с фото 'ДО'

#### Качество фотографий
- Хорошее освещение
- Четкое изображение
- Весь объем работ в кадре
- Без фильтров и обработки

### 💳 Финансовые операции

#### Мои реквизиты
1. Нажмите "💳 Мои реквизиты"
2. Вы увидите текущие данные:
   - 📞 Телефон: ваш номер
   - 🏦 Банковские данные: реквизиты для оплаты

#### Обновление реквизитов
1. Нажмите "/update_requisites"
2. Введите ваш номер телефона
3. Введите банковские данные:
   - Номер карты
   - Или полные реквизиты счета
4. Сохраните изменения

#### Получение оплаты
1. Выполните заказ и загрузите фото
2. Менеджер проверит работу
3. Менеджер создаст платеж
4. Оплата поступит на ваши реквизиты
5. Вы получите уведомление о платеже

## 🔄 Процесс работы над заказом

### 📋 Шаг 1: Выбор заказа
- Изучите доступные заказы
- Обратите внимание на:
  - Местоположение
  - Время выполнения
  - Тип уборки
  - Стоимость

### 🤝 Шаг 2: Взаимодействие
- Свяжитесь с клиентом перед визитом
- Уточните детали и требования
- Подтвердите время прибытия

### 🧹 Шаг 3: Выполнение работ
- Прибыть вовремя
- Сделать фото 'ДО'
- Выполнить уборку качественно
- Сделать фото 'ПОСЛЕ'

### 📸 Шаг 4: Документация
- Загрузить оба фото в систему
- Проверить качество изображений
- Убедиться что фото загружены

### 💰 Шаг 5: Получение оплаты
- Дождаться проверки менеджером
- Получить подтверждение выполнения
- Получить оплату на реквизиты
