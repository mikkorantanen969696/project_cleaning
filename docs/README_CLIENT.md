# 📋 Инструкция для Клиента

## 🎞️ Как заказать услугу уборки

### 📞 Связь с менеджером
1. Найдите нас через:
   - 📱 Telegram
   - 📞 Телефон
   - 🌐 Авито и другие площадки
2. Свяжитесь с менеджером
3. Обсудите детали заказа
4. Менеджер создаст заказ в системе

### 📋 Процесс заказа
1. **Консультация** - менеджер уточнит все детали
2. **Создание заказа** - внесение информации в систему
3. **Подтверждение** - вы получите подтверждение заказа
4. **Назначение клинера** - система найдет исполнителя
5. **Выполнение** - клинер прибудет в указанное время
6. **Оплата** - после выполнения работ

## 💰 Оплата услуг

### 📄 Способы оплаты
1. **QR-код** - быстрый способ через банковское приложение
2. **Счет на оплату** - PDF документ с реквизитами
3. **Наличные** - при согласовании с менеджером
4. **Банковский перевод** - по реквизитам из счета

### 📧 Процесс оплаты
1. Менеджер отправит вам счет
2. Вы получите QR-код для быстрой оплаты

## 🧹 Виды услуг

### 🧹 Регулярная уборка
- Поддерживающая уборка помещений
- Удаление пыли и грязи
- Мытье полов и поверхностей
- Вывоз мусора

### 🧼 Глубокая уборка
- Тщательная очистка всех поверхностей
- Мытье труднодоступных мест
- Обработка мебели и техники
- Дезинфекция помещений

### 🏗️ Послестроительная уборка
- Удаление строительной пыли
- Сбор строительного мусора
- Мытье окон и поверхностей
- Подготовка к заселению

### 🪟 Уборка окон
- Мытье окон с обеих сторон
- Очистка рам и подоконников
- Удаление загрязнений
- Полировка стекол

## 📝 Что нужно знать

### 📋 Перед визитом клинера
1. **Подготовьте помещение** - уберите личные вещи
2. **Обеспечьте доступ** - клинер должен свободно перемещаться
3. **Укажите особенности** - сообщите о ценных вещах
4. **Подготовьте оплату** - если оплата наличными

### 🏠 Во время уборки
1. **Будьте доступны** - для уточнения деталей
2. **Проверьте результат** - после завершения работ
3. **Подпишите акт** - при необходимости
4. **Оплатите услуги** - по факту выполнения

### 📸 Фотофиксация
- Клинер делает фото 'ДО' начала работ
- Клинер делает фото 'ПОСЛЕ' завершения
- Фото служат подтверждением качества
