# 📋 Инструкция для Администратора

## 🔐 Первоначальная настройка

### 1. Установка и запуск
1. Склонируйте репозиторий
2. Создайте виртуальное окружение: `python -m venv venv`
3. Активируйте: `venv\Scripts\activate` (Windows) или `source venv/bin/activate` (Linux/Mac)
4. Установите зависимости: `pip install -r requirements.txt`
5. Настройте `.env` файл (см. `.env.example`)
6. Инициализируйте базу данных: `python scripts/init_database.py`
7. Запустите бота: `python bot/main.py`

### 2. Настройка переменных окружения (.env)
```env
BOT_TOKEN=your_telegram_bot_token_here
DATABASE_URL=postgresql+asyncpg://username:password@localhost:5432/project_cleaning
ADMIN_TELEGRAM_ID=your_admin_telegram_id
ADMIN_USERNAME=your_admin_username
```

## 🎯 Основные функции Администратора

### 👥 Управление пользователями

#### Добавление менеджера
1. Нажмите "👥 Управление пользователями"
2. Выберите "➕ Добавить менеджера"
3. Введите username менеджера (без @)
4. Система сгенерирует пароль
5. Отправьте пароль менеджеру
6. При необходимости введите Telegram ID менеджера

#### Удаление/деактивация менеджера
1. Нажмите "👥 Управление пользователями"
2. Выберите "➖ Удалить менеджера"
3. Выберите менеджера из списка
4. Подтвердите деактивацию

### 📊 Статистика и отчеты

#### Просмотр общей статистики
- **Всего заказов** - общее количество созданных заказов
- **Выполнено** - количество успешно выполненных заказов
- **Отменено** - количество отмененных заказов
- **Процент выполнения** - соотношение выполненных к общим
- **Общая выручка** - сумма всех выполненных заказов
- **Средний чек** - средняя стоимость заказа

#### Статистика по менеджерам
- Количество заказов каждого менеджера
- Процент выполнения
- Общая выручка
- Сравнительная таблица

#### Экспорт данных
- **Excel** - полная статистика в таблицах
- **PDF** - форматированные отчеты
- **CSV** - сырые данные для анализа

### 🏙️ Управление городами

#### Добавление города
1. Нажмите "🏙️ Управление городами"
2. Введите название города
3. Укажите ID Telegram темы для заказов

#### Настройка тем в Telegram
- Создайте супергруппу
- Добавьте подтемы для каждого города
- Получите ID подтем и добавьте в систему

### 💳 Финансы

#### Управление платежами
- Просмотр всех платежей
- Подтверждение оплат
- Экспорт финансовых отчетов

## 🔧 Технические функции

### 📋 Создание заказов
Администратор может создавать заказы так же, как и менеджер:
1. Нажмите "📝 Создать заказ"
2. Заполните все поля:
   - Имя клиента
   - Телефон
   - Адрес
   - Город
   - Тип уборки
   - Дата и время
   - Продолжительность
   - Стоимость
   - Оборудование и средства
   - Примечания

### 📊 Детальная статистика

#### По периодам
- Выберите диапазон дат
- Получите статистику за период
- Сравните с другими периодами

#### По городам
- Активность в каждом городе
- Популярные услуги
- Средние чеки

#### По клинерам
- Количество выполненных заказов
- Рейтинг выполнения
- Доходы клинеров

## 🚨 Устранение проблем

### Частые проблемы

#### Бот не запускается
1. Проверьте токен в `.env`
2. Убедитесь что база данных доступна
3. Проверьте права доступа к файлам

#### Менеджер не может авторизоваться
1. Проверьте что пароль правильный
2. Убедитесь что менеджер не деактивирован
3. Проверьте Telegram ID

#### Заказы не появляются в теме города
1. Проверьте ID темы города
2. Убедитесь что бот имеет права администратора
3. Проверьте что тема активна

### 📞 Поддержка
При возникновении проблем:
1. Проверьте логи бота
2. Проверьте подключение к базе данных
3. Свяжитесь с разработчиком

## 🔄 Резервное копирование

### База данных
```bash
pg_dump project_cleaning > backup.sql
```

### Восстановление
```bash
psql project_cleaning < backup.sql
```
