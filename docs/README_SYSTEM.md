# 📋 Общее описание системы

## 🎞️ О системе

Система управления клининг-бизнесом - это комплексное решение для автоматизации процессов уборочных услуг через Telegram бот. Система предназначена для взаимодействия трех ключевых ролей: администраторов, менеджеров и клинеров.

## 👥 Роли в системе

### 🔐 Администратор
- **Полный контроль** над системой
- **Управление пользователями** - добавление/удаление менеджеров
- **Статистика** - полная аналитика по всем аспектам бизнеса
- **Финансы** - контроль всех платежей
- **Настройки** - конфигурация городов и параметров системы

### 💼 Менеджер
- **Создание заказов** - прием заявок от клиентов
- **Коммуникация** - взаимодействие с клиентами и клинерами
- **Финансы** - выставление счетов и оплата клинеров
- **Статистика** - ограниченная аналитика по своей работе
- **Контроль качества** - проверка выполненных работ

### 🧹 Клинер
- **Поиск заказов** - выбор доступных заказов в своем городе
- **Выполнение работ** - качественное оказание услуг
- **Фотофиксация** - доказательство выполнения работ
- **Получение оплаты** - своевременное вознаграждение
- **Репутация** - построение рейтинга надежности

## 🔄 Бизнес-процесс

### 📞 Входящий запрос
1. Клиент обращается через любой канал
2. Менеджер обрабатывает заявку
3. Создается заказ в системе

### 📋 Назначение исполнителя
1. Заказ появляется в теме города
2. Клинеры видят доступные заказы
3. Клинер берет заказ
4. Менеджер получает уведомление

### 🏠 Выполнение работ
1. Клинер связывается с клиентом
2. Выполняет уборку в указанное время
3. Делает фото "ДО" и "ПОСЛЕ"
4. Загружает фото в систему

### 💰 Оплата
1. Менеджер проверяет качество работ
2. Выставляет счет клиенту
3. Клиент оплачивает услуги
4. Менеджер оплачивает клинеру

## 📈 Метрики успеха

### 📊 Операционные KPI
- **Время создания заказа** - < 2 минуты
- **Время назначения клинера** - < 30 минут
- **Процент выполнения заказов** - > 95%
- **Среднее время выполнения** - по стандарту

### 💰 Финансовые KPI

- **Рентабельность** - > 30%

### 🎞️ Клиентские KPI
