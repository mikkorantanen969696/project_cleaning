# 🧹 Система управления клининг-бизнесом

## 📋 Описание проекта

Комплексная система для автоматизации клининг-бизнеса через Telegram бот с ролевой моделью доступа (администратор, менеджер, клинер).

### 🎯 Основные возможности

- **🔐 Ролевая система доступа** с разными правами для пользователей
- **📝 Автоматическое создание заказов** с детальной информацией
- **🤖 Telegram бот** как основной интерфейс взаимодействия
- **📊 Детальная статистика** и отчеты по всем аспектам бизнеса
- **💰 Финансовый контроль** с автоматическими счетами и платежами
- **📸 Фотофиксация** выполненных работ
- **🏙️ Географическая сегментация** по городам

### 👥 Роли в системе

#### 🔐 Администратор
- Управление пользователями (добавление/удаление менеджеров)
- Полная статистика по бизнесу
- Управление городами и темами
- Финансовый контроль
- Экспорт данных

#### 💼 Менеджер
- Создание заказов от клиентов
- Выставление счетов в PDF формате
- Оплата услуг клинеров
- Просмотр ограниченной статистики
- Коммуникация с клиентами и клинерами

#### 🧹 Клинер
- Поиск и принятие заказов
- Выполнение работ с фотофиксацией
- Получение оплаты
- Управление реквизитами
- Рейтинговая система

## 🚀 Быстрый старт

### 📋 Требования
- Python 3.11+
- PostgreSQL 12+
- Telegram Bot Token
- ОС: Windows/Linux/macOS

### ⚙️ Установка

1. **Клонирование репозитория**
```bash
git clone <repository_url>
cd project_cleaning
```

2. **Создание виртуального окружения**
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

3. **Установка зависимостей**
```bash
pip install -r requirements.txt
```

4. **Настройка переменных окружения**
```bash
cp .env.example .env
# Отредактируйте .env файл с вашими данными
```

5. **Инициализация базы данных**
```bash
python scripts/init_database.py
```

6. **Запуск бота**
```bash
python bot/main.py
```

## 🔧 Конфигурация

### 📄 Переменные окружения (.env)

```env
# Telegram Bot Token
BOT_TOKEN=your_telegram_bot_token_here

# Database Configuration
DATABASE_URL=postgresql+asyncpg://username:password@localhost:5432/project_cleaning

# Admin Configuration
ADMIN_TELEGRAM_ID=your_admin_telegram_id
ADMIN_USERNAME=your_admin_username

# Payment Configuration (optional)
PAYMENT_PROVIDER_TOKEN=your_payment_provider_token

# File Storage
PHOTOS_PATH=./photos
PDF_PATH=./pdfs

# Logging
LOG_LEVEL=INFO
LOG_FILE=./logs/bot.log
```

### 🗄️ Настройка базы данных

#### Локальная разработка
```bash
# Создание базы данных
createdb project_cleaning

# Настройка пользователя
psql -d project_cleaning -c "CREATE USER cleaning_user WITH PASSWORD 'password';"
psql -d project_cleaning -c "GRANT ALL PRIVILEGES ON DATABASE project_cleaning TO cleaning_user;"
```

#### TimeWeb хостинг
```env
DATABASE_URL=postgresql+asyncpg://username:password@host:port/database_name
```

## 📊 База данных

### 🏗️ Основные таблицы

- **users** - Пользователи системы (администраторы, менеджеры, клинеры)
- **cities** - Города и их Telegram темы
- **orders** - Заказы на уборку
- **order_photos** - Фото "ДО" и "ПОСЛЕ"
- **payments** - Платежи и финансовые операции
- **statistics** - Агрегированная статистика

### 🔄 Связи между таблицами

```
users (1:N) orders (manager_id)
users (1:N) orders (cleaner_id)
cities (1:N) orders
orders (1:N) order_photos
orders (1:N) payments
```
